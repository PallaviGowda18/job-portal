# Job Portal Web Application

This is a Full Stack Job Portal developed using Node.js and MongoDB.

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- Node.js
- Express.js
- MongoDB

## Features
- User Registration
- Login System
- Dashboard
- Post Jobs
- View Jobs
- Apply Jobs
- Job Search

## Project Structure

frontend/
backend/
server.js
package.json

## How to Run the Project

1. Install dependencies
npm install

2. Start server
node server.js

3. Open frontend files in browser
